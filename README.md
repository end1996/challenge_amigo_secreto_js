## Run Locally

Clone the project

```bash
  git clone https://github.com/end1996/challenge_amigo_secreto_js.git
```

Go to the project directory

```bash
  cd challenge_amigo_secreto_js
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
